##  Referensi Desain & Implementasi

### Design Inspiration
- **https://www.behance.net/search/projects/barbershop** - Dark theme dengan gold accent (#d4af37), card-based layout untuk team/gallery, modified color palette

- **https://dribbble.com/search/barbershop** - Modern hover effects, glassmorphism design, gradient overlays, border styling

### Teknologi & Framework
- **https://getbootstrap.com/docs/5.3/** - Responsive navbar toggler, carousel, forms, custom component styling

- **https://fonts.google.com/** - Poppins font (modern, clean, profesional) dengan letter-spacing

- **https://coolors.co/** - Gold (#d4af37) primary accent color untuk branding dengan dark background

### CSS & Animations
- **https://animate.style/** - Custom keyframe animations (glow, slide transitions, smooth micro-interactions)

- **https://www.w3schools.com/css/css_rwd_intro.asp** - Mobile-first responsive design, flexbox/grid layout, breakpoints

- **https://neumorphism.io/** - Glassmorphism effects untuk testimonial cards

##  Tech Stack
- **HTML5** - Semantic structure
- **CSS3** - Grid, Flexbox, Gradients, Animations, Keyframes
- **JavaScript (jQuery)** - Form validation, dark mode, search filter, data persistence
- **Bootstrap 5.3.3** - Responsive framework
- **localStorage API** - Data persistence (contact form)

---

##  Fitur Utama
 Responsive Navbar (hamburger menu mobile)  
 Dark/Light Mode toggle (persistent)  
 Service Search (real-time filter)  
 Contact Form + Data Table  
 Testimonial Carousel  
 Gallery Grid responsive  
 Smooth animations & hover effects  
 Mobile-first design

##  File Structure
```
├── index.html          # Main HTML + semantic structure
├── style.css           # Complete CSS styling + responsive
├── script.js           # jQuery + localStorage functionality
└── images              # Team & gallery images
```

---

##  Cara Menggunakan
1. Buka `index.html` dengan browser
2. Navigasi menggunakan navbar
3. Toggle dark mode dengan tombol di bawah
4. Isi contact form untuk simpan data pelanggan
5. Data tersimpan di browser storage (persistent)

---

## Responsive
- **Desktop** (>991px) - Full navbar, multi-column
- **Tablet** (768-991px) - Navbar toggler, 2-column  
- **Mobile** (<768px) - Hamburger menu, single column
